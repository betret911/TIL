# 파이썬 기초

- `input()` : 키보드로 입력한 값을 가져 옴
- `.split()` : () 안 요소를 기준으로 입력된 값을 나눔
- `print(?, ? sep='')` : sep, ''안의 기호를 사이에 두고 값을 출력
- `s[a:b]` : a번째 문자부터 b-1번째 문자까지 자름
- `ord()` : 유니코드 정수 파싱
- `chr()` : 유니코드, 정수값 -> 문자
- `print(문자열*정수)` : 문자열을 정수만큼 반복해 출력
- `a**b` : 거듭제곱 연산자
- `format(수, ".2f")` : 원하는 자리까지 반올림 된 실수
